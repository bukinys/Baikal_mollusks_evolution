# Baikal_mollusks_evolution
The repository contains primary data in the form of decoded nucleotide sequences (sequences of nuclear (ITS1, ATPSα-intron) and mitochondrial (COI) markers) for Baicalia endemic mollusks of the genus Baicalia and data on program settings for evolutionary analysis.

Baicalia_COI.fasta - nucleotide sequences of the mitochondrial genetic marker COI (cytochrome c oxidase subunit 1, COI)

Baicalia_ATPSa.fasta - nucleotide sequences of the nuclear genetic marker ATPSα (intron of ATP synthase subunit alpha, ATPSα)

Baicalia_ITS.fasta - nucleotide sequences of the nuclear genetic marker ITS1 (internal transcribed spacer 1, ITS1)

Tree_COI.sh - settings for running the IQTREE program for testing the nucleotide substitution model and tree reconstruction with topology reliability assessment using the COI marker

Tree_ATPS.sh - settings for running the IQTREE program for testing the nucleotide substitution model and tree reconstruction with topology reliability assessment using the ATPSα marker

Tree_ITS_sub.sh - settings for running the IQTREE program for testing the nucleotide substitution model and tree reconstruction with topology reliability assessment using the ITS1 marker
